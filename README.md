# Listerine

## Getting Started

Rename [config.exs.example](config/config.exs.example) to `config.exs` and insert your Token.

#### Get the dependencies
```shell
mix deps.get
```

#### Compile
```shell
mix compile
```

#### Run
```shell
mix run --no-halt
```
